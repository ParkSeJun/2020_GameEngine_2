# 1. 게임 개요   
### 기획 컨셉   
    수업에서 배운 유니티 기능들을 잘 활용하여 간단한 게임을 구현한다.
### 장르  
    캐주얼 FPS
### 이용 가능 연령  
    전 연령  
### 게임 플레이 방법  
    주어진 무기를 이용해 몬스터를 물리치거나 피해서 목적지까지 이동
### 조작 방법  
* A, W, S, D : 이동  
* 마우스 우클릭 : 조준
* 마우스 좌클릭 : 공격
